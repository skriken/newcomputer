## newcomputer

Simple scripts to setup the most used applications and tools for either gaming or work.

1. Open PowerShell

2. Prepare

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

3. Execution

Work:
```
.\base.ps1
.\work.ps1
```

Gaming:
```
.\base.ps1
.\gaming.ps1
```
